Project Overview - Justin Owens


This application allows users to log their hours and mission work activity done either on their own or via their church. The application allows the user to enter and update their personal information as well as being able to log in and out of the application for security. The user can also clear all of their work history and start from scratch.


The portion of this project that challenged me the most was getting the JSON data stored and read from when displaying the data. I had worked with objects before, but the entries were stored in a multi-dimensional array and, once I figured that out, I could display it properly. I was also challenged with having two functions load concurrently when the page loads to display both user data and work activity data.
