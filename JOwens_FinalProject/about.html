Project Overview - Justin Owens


This application allows missionaries to log their fundraising efforts by noting the name of the church/person who gave, the date of the giving, the amount given, and the monthly amount pledged toward their monthly funding goal. The application allows the user to enter and update their personal information as well as create a username and password to be authenticated via Google Firebase. The user can also clear all of their funding history and start from scratch for a new fundraising project. On a separate page, the information entered by the user for their fundraising is displayed in a graph noting the date and amount given.


To begin using the application, the user must enter the temporary username of temp@temp.com and temporary password of "000000." After a profile is created, the user can use their own created username and password to login.
